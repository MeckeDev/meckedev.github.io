User Commands

Command Parameters Example Description
$commands None $commands Returns a list of all Custom-Commands on the Channel
Output: $discord, $test, $insta
$roll_ds weight $roll_ds 25.5 Returns the Weightlimits for different Rolls in Dark Souls 1
Output: Fast: 6.38 | Mid: 12.75 | Fat: 25.50
$memeroll weight $memeroll 25.5 Returns the Range of Weight you need for a Speedrun-Trick in Dark Souls 1
Output: Meme-Roll/Air-Roll: between 6.38 - 7.42
$rps rock, paper, scissors $rps paper Play Rock Paper Scissors against the Bot
Output: That's a Draw, the Bot also picked Paper

Streamer/Mod-Only Commands


Custom-Commands can include -KEYWORDS- like described down below

Command Parameters Example Description
$addcom commandname, value $addcom test This is a test creates a Command with the given commandname and outputs the Value
Output: "This is a test" was saved as $test.
$add_counter countername, value $add_counter test This test was done
-COUNT- times.		 creates a Counter with the given commandname and outputs the Value
Output: "This is a test" was saved as $test.
$mod_counter countername, value $mod_counter test This test was done
-COUNT- times.		 creates a Mod-Only Counter with the given commandname and outputs the Value
(Can be called by Everyone, but can only be changed by Mods)
Output: "This is a test" was saved as $test.
$subcom commandname, value $subcom test This is a test creates a Sub-Only Command with the given commandname and outputs the Value
Output: "This is a test" was saved as $test (only Subs).
$modcom commandname, value $modcom test This is a test creates a Mod-Only Command with the given commandname and outputs the Value
Output: "This is a test" was saved as $test (only Mods).
$remcom commandname $remcom test removes an existing Command with the given commandname
Output: $test was removed.

Mecke-Only Commands

Command Parameters Example Description
$run oneline Python $run [x for x in range(5)] Runs Python Code and returns Result
Output: [0, 1, 2, 3, 4]
$join channelname $join mecke_dev Bot joins a new Channel
Output: I joined the Channel: mecke_dev
$restart None $restart Restarts the Bot
Output: I will be back soon

-KEYWORDS-

Keyword Parameter Example Description
-COUNT- +, -, set, reset, show This Counter was used -COUNT- times / $test set 120 Returns the Counter with the current Value
Output: This Counter was used 120 times
-VIEWER- None I would say -VIEWER- is the best gets Replaced with a Random Name from the current Viewerlist
Output: I would say @chris_live is the best
-TIME- None Only -TIME-25-05-2050-00-00-00- until the World ends. gets replaced by a countdown until a specific Time
Output: Only 28 Years 5 Hours 26 Minutes 43 Seconds until the World ends.
-RNG- None We found -RNG-0-1000- Bugs in the Bot. gets replaced by a random Number between the given Numbers
Output: We found 268 Bugs in the Bot.
-ITEM- itemname
(can be multiple Words seperated by _ )		 The Streamer needs -ITEM- to create Content. / $test an_Airplane gets replaced by whaterever the User wants
Output: The Streamer needs an Airplane to create Content.
